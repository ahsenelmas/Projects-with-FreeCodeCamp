Python

Python is an interpreted, high-level language.

It emphasizes readability and simplicity.

print("Hello, World!")

Python uses indentation to define blocks.

  • Easy to learn
  • Popular for AI
Java

Java is an object-oriented programming language.

It runs on the JVM (Java Virtual Machine).

System.out.println("Hello, Java!");
  • Platform independent
  • Used in enterprise applications
Javascript

JavaScript powers dynamic web content.

It runs in the browser.

document.getElementById("demo").innerHTML = "JS!";
  • Event-driven
  • Client-side scripting
C++

C++ is an extension of C.

It supports both procedural and object-oriented programming.

std::cout << "Hello, C++!" << std::endl;
  • Fast and efficient
  • Widely used in game development
HTML

HTML is the standard markup language for web pages.

It structures content using elements and tags.

<h1>Hello HTML</h1>
  • Semantic tags
  • Base of web development
